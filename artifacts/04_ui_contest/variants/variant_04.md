# UI 方案变体 4：自然森系 Natural Forest

## 1. 方案名称与定位

| 属性 | 内容 |
|------|------|
| 方案名称 | 自然森系 Natural Forest |
| 核心关键词 | 有机形态、大地色系、木纹质感、绿植元素、舒缓呼吸感 |
| 适合门店类型 | 森系自然风拼豆店、主打环保理念的手作工作室、开在植物园/公园附近的文艺门店、强调"慢生活"和"自然材料"的店铺 |
| 适合用户群 | 20-35 岁注重生活方式和环保理念的用户，喜欢植物、日系杂货、自然材质，追求"逃离城市喧嚣"的手作体验 |
| 设计理念 | 将拼豆体验比喻为"在森林里捡拾彩色果实，拼出心中的图案"。用大地色系（泥土棕、苔藓绿、树皮灰）构建沉稳基调，点缀浆果红和花朵黄。大量使用有机形态（非完美圆形、不规则圆角）、木纹和叶片纹理元素。整体氛围像走进一间被植物包围的木屋工作室。 |

---

## 2. 视觉风格关键词

大地色调、有机曲线、木纹肌理、植物插画、不对称圆角、低饱和自然色、日系杂货风、露营美学

---

## 3. 设计系统 Design Tokens

### 3.1 色彩系统

从森林景观提取配色：泥土、苔藓、树皮、浆果、蘑菇。

| 角色 | 色值 | 灵感来源 | 用途 |
|------|------|----------|------|
| **主色-树皮棕** | `#6B5B4B` | 树干树皮 | 主按钮、重点文字、导航 |
| **主色-苔藓绿** | `#5B7B5E` | 森林苔藓 | 成功状态、计时正常、确认操作 |
| **点缀-浆果红** | `#C5655B` | 森林浆果 | 危险操作、计时紧急、重要提醒 |
| **点缀-蘑菇黄** | `#D4A853` | 野生蘑菇 | 待确认状态、温和提示 |
| **背景色-米白** | `#FBF7F2` | 未漂白棉纸 | 全局页面底色（微暖米白） |
| **卡片-亚麻白** | `#FDFAF5` | 亚麻布料 | 卡片背景（比底色稍暖） |
| **表面-木色** | `#F1EBE3` | 浅色木纹 | 次级表面、选中态背景 |
| **边框-藤色** | `#E0D7CB` | 干藤蔓 | 分割线、卡片边框 |
| **主文字-墨绿棕** | `#2E2B26` | 深色树皮 | 标题、正文 |
| **辅文字-灰绿** | `#7A756B` | 灰绿色叶片 | 辅助说明 |
| **禁用-浅棕** | `#BFB8AD` | 枯叶 | 占位符、禁用态 |

**状态色体系：**

| 状态 | 色值 | 自然联想 |
|------|------|----------|
| 待确认 | `#D4A853` 蘑菇黄 | 等待成熟的蘑菇 |
| 已确认 | `#5B7B5E` 苔藓绿 | 生机勃勃的绿植 |
| 已拒绝 | `#C5655B` 浆果红 | 熟透落地的浆果 |
| 已完成 | `#7B8B8A` 湖水灰绿 | 沉静的湖水 |
| 已取消 | `#BFB8AD` 枯叶色 | 自然凋落的叶片 |
| 计时预警 | `#D4A853` 蘑菇黄 | 傍晚的色调 |
| 计时紧急 | `#C5655B` 浆果红 + 树叶飘落动画 | 秋叶飘零 |

### 3.2 字体层级

小程序端使用更柔和的字体风格：

| 层级 | 字重 | 字号 | 说明 |
|------|------|------|------|
| H1 | SemiBold (600) | 38rpx | 页面主标题，略带衬线感的系统字体 |
| H2 | SemiBold (600) | 32rpx | 区块标题 |
| H3 | Medium (500) | 28rpx | 卡片标题 |
| Body | Regular (400) | 28rpx | 正文，行高 1.6 倍（增加呼吸感） |
| Caption | Regular (400) | 24rpx | 辅助文字，行高 1.5 倍 |
| Small | Medium (500) | 22rpx | 标签、徽章 |

Web 后台：

| 层级 | 字重 | 字号 |
|------|------|------|
| H1 | SemiBold (600) | 22px |
| H2 | SemiBold (600) | 17px |
| H3 | Medium (500) | 15px |
| Body | Regular (400) | 14px |
| Caption | Regular (400) | 12px |

### 3.3 间距系统

基础单位：8rpx，但使用更大的区块间距营造"森林中的开阔感"。

| Token | 值 | 用途 |
|-------|------|------|
| xs | 8rpx | 紧凑间距 |
| sm | 16rpx | 常规内边距 |
| md | 24rpx | 卡片内边距 |
| lg | 36rpx | 页面左右页边距 |
| xl | 56rpx | 区块间隔 |
| xxl | 80rpx | 大区块间隔、"森林呼吸感"来源 |

### 3.4 圆角规则

采用"有机圆角"——避免完美对称的圆角值，使用不同的圆角组合。

| 元素 | 圆角值 | 说明 |
|------|--------|------|
| 卡片 | 左上 24rpx / 右上 16rpx / 右下 24rpx / 左下 12rpx | 不对称圆角，模拟自然形态 |
| 较小卡片 | 16rpx / 20rpx / 12rpx / 20rpx | 不对称 |
| 主按钮 | 24rpx（全对称，因为按钮需要稳定感） | 大圆角但非胶囊形 |
| 标签/徽章 | 16rpx（大圆角） | 像圆润的鹅卵石 |
| 输入框 | 12rpx | 柔和微圆角 |
| 弹窗 | 左上 32rpx / 右上 32rpx / 0 / 0 | 仅顶部大圆角 |
| 图片容器 | 12rpx | 微圆角 |

### 3.5 阴影层级

阴影带暖色调，模拟阳光透过树叶在地面投下的影子。

| 层级 | 值 |
|------|-----|
| L1 轻阴影 | `0 2rpx 16rpx rgba(107,91,75,0.06)` — 暖棕调阴影 |
| L2 中阴影 | `0 6rpx 24rpx rgba(107,91,75,0.10)` |
| L3 重阴影 | `0 10rpx 40rpx rgba(107,91,75,0.16)` |

### 3.6 纹理系统（纹理通过 CSS 实现，避免图片资源）

| 纹理 | 实现方式 | 用途 |
|------|----------|------|
| 纸张纹理 | `background: #FDFAF5 + subtle noise SVG` | 卡片背景，模拟手工纸 |
| 木纹点缀 | `linear-gradient(90deg, rgba(107,91,75,0.05) 1px, transparent 1px)` 重复 | 侧边栏、顶部条 |
| 叶片装饰 | CSS 绘制的简化叶片形状（::before/::after 伪元素） | 页面标题旁点缀 |

## 6. 交互细节

### 6.1 页面转场

- 小程序使用微信默认转场
- 弹窗：从底部生长出来（scaleY: 0→1，从中心向上展开），像植物破土。时长 350ms，带 ease-out 缓动
- 遮罩：fade-in 200ms

### 6.2 关键反馈动画

- **预约成功**：全屏闪现极淡的绿色（opacity 0.1），同时主按钮上冒出三片小叶子飘散动画（CSS animation: 向上飘+旋转+淡出，800ms）
- **确认预约**：卡片左侧的彩色竖条从蘑菇黄平滑过渡为苔藓绿（color transition 400ms）
- **计时紧急（剩余<5分钟）**：倒计时数字颜色从苔藓绿→蘑菇黄→浆果红渐变（不是切换，是平滑过渡）。同时卡片背景出现极淡的暖色脉冲（opacity 变化 0→0.05→0，周期 4s，缓慢不急促）
- **加时操作**：进度条向右生长一段（宽度增加动画 300ms ease-out），同时冒出 🌱 小苗图标
- **结束计时**：卡片缩小+透明度降低，同时卡片中的进度条走满到 100% 然后整个卡片像落叶一样向下飘出（translateY + rotate + opacity），时长 500ms

### 6.3 手势操作

- 预约卡片左滑：露出操作按钮时，按钮从卡片"土壤"中"生长"出来（scaleX: 0→1 origin left）
- 下拉刷新：顶部出现叶片旋转加载指示器
- 计时卡片支持双指缩放查看详情（V1 简化为点击进入）

## 7. 优点

1. **极度差异化**。自然森系在工具类小程序中极为罕见，品牌辨识度极高，一眼记住。
2. **情感连接强**。自然隐喻系统（蘑菇=待确认，苔藓=正常，浆果=紧急）比抽象的颜色编码更人性化，降低学习成本。
3. **减压氛围**。大地色系 + 有机形态在视觉上天然具有舒缓效果，与拼豆手作的"逃离压力"定位完美契合。
4. **叙事性强**。每个页面都在讲一个"森林里的手作故事"，品牌内容营销素材丰富。
5. **可访问性优秀**。图标+颜色双重编码状态，色盲用户不影响使用。
6. **天然适合植物/环保主题扩展**。未来可自然延伸至植物养护、环保材料等品类。

## 8. 缺点

1. **Web 后台效率可能受影响**。自然隐喻和有机形态在数据密集的后台表格中可能显得不够"严肃"，商家初次使用可能需要适应。
2. **图标依赖度高**。自然图标隐喻（🍄🌿🍂）需要用户学习对应关系，虽然比抽象颜色直观，但仍有认知成本。
3. **实现难度较高**。不对称圆角、有机形态、纹理背景、叶片动画等效果需要较多 CSS 技巧，在 uni-app 和微信小程序环境中的还原度可能受限。
4. **不适合所有场景**。如果门店实际环境是工业风/科技风，这个设计会显得违和。
5. **图片资源可能较多**。植物插画、纹理等装饰元素可能增加小程序包体积（需控制在 2MB 内）。

## 9. 实现难度

**中等偏高**。不对称圆角需要独立声明每个角的 border-radius；CSS 纹理需要精细调整；植物动画需要 animation keyframes；自然图标系统需要建立完整的组件映射。Element Plus 默认组件需要较多样式覆盖。

## 10. 可复用性

**中等**。自然森系的 Design Token 体系可以复用到其他"自然/环保/生活方式"品类（如植物店、咖啡馆、书店），但需要针对具体品类调整自然隐喻系统（如咖啡馆用"咖啡豆→研磨→萃取"替代"蘑菇→苔藓→浆果"）。
