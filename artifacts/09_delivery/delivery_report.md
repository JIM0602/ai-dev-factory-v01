## 1. 项目概况

| 项目 | 内容 |
|------|------|
| **项目名称** | 拼豆店预约计时小程序 |
| **一句话描述** | 面向拼豆店个体商家的双端微信小程序工具，核心解决门店容量管理、预约管理、到店计时和会员记录沉淀问题。 |
| **产品定位** | 轻量级 MVP，单门店单商家模式，聚焦预约与计时核心流程。 |
| **技术栈** | 小程序：uni-app + Vue 3 + TypeScript；Web 后台：Vue 3 + Vite + TypeScript + Element Plus；后端：NestJS + TypeScript + TypeORM；数据库：PostgreSQL 15；部署：Docker Compose |
